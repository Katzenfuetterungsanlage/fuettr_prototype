# Übertragungsprotokoll

## 1.1 Maschinenstatus

| last_time | next_time | next_time_in | machine_state |
|-----------|-----------|--------------|---------------|
|   string  |  string   |    string    |    string     |

Beispiel: {  
    "last_time": "10:10",  
    "next_time": "11:11",  
    "next_time_in": "10min"  
}

## 1.2 Fütterungszeiten

| time1  | time2  | time3  | time4  |
|--------|--------|--------|--------|
| string | string | string | string |

Beispiel: {  
    "time1": "10:10",  
    "time2": "11:11",  
    "time3": "12:12",  
    "time4": "--:--"  
}
